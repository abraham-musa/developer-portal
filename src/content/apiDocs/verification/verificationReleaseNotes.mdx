## November 8, 2018
Launched the Disability Rating API
- Disability Rating API endpoint for external API consumers. Retrieves disability rating of an authorized Veteran.

## November 8, 2018
Launched the Veteran Confirmation API
- Allows third-parties to request confirmation from the VA of an individual's Veteran status after receiving authorization to do so using an Open ID Connect flow.

## June 11, 2018 
Expose Service History via API gateway as POC [#172](https://github.com/department-of-veterans-affairs/vets-contrib/issues/172)
- Given an authenticated user, this created an API endpoint that returns their service history resolved from eMISe
